# Product

Imagine MCP. A Model Context Protocol server for ImagineArt that exposes eight creative endpoints (text-to-image, text-to-video, music, scripts, upscaler, background remover, list, balance) to Claude, Cursor, Cline, Windsurf, and any MCP-compatible client. Built by Vyro AI. Bridges the imagine.art creative API to AI agents. Not open source — distributed as a hosted/published package, sign-in via imagine.art account.

# Audience

Developers building with AI agents. Comfortable with terminals, `npx`, MCP server configs, and JSON. They install MCP servers and read READMEs. They are not the target of pitch decks or "endless possibilities" marketing.

# Register

`brand`. The landing page is the product's first impression for developer-evaluators deciding whether to install. Marketing is the design surface, not in-product UI.

# Tone

Plain, technical, unhurried. Sentences end in periods. No exclamation points, no rhetorical questions, no "endless possibilities", no "supercharge". Reads like a thoughtfully-written README, not a startup pitch.

# Anti-references (must not look like)

- Generic AI SaaS landing: gradient blob backdrop, glowing CTA, gradient text, em-dashes everywhere, pulsing dots, infinite-scroll logo marquees, animated counters, hero-metric stat cards.
- Higgsfield / image-gen-product clones with rainbow purple/pink wash and glassmorphism.
- "Eight tools. One protocol." style gradient-word taglines.

# Brand

- Brand color: `#8A3FFC` (`--primary-60`). Purple does the heavy lifting; pink is no longer used. Secondary accent `#FB5607` (orange) appears in imagine.art's production palette and is used sparingly here (e.g., script scene heading).
- Display + Body / UI: Google Sans Flex (variable axis 100–900). Same family imagine.art uses across hero, headings, body, and UI.
- Code surfaces use the system monospace stack (`ui-monospace`, `SF Mono`, `Menlo`) — matches imagine.art's restraint of not loading a developer-mono webfont.
- Section eyebrows are short mono uppercase labels (`v1.0`, `Toolset`, `Works with`, `Ship it`) — the consistent typographic device that ties sections together.

# Strategic principles

1. **Show, don't sell.** The eight tools are the product. Each card has a calm, static visual. No fake animations to manufacture energy.
2. **Developers test before they trust.** The install path (method tabs + 3 numbered steps + copy-button) lives in the hero, not buried below.
3. **Less is more credible.** On a dev-tool page every glow, blob, and gradient subtracts trust. Hairline borders and typography do the structural work.
