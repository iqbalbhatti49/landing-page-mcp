Creative tools
for your agent

The Model Context Protocol server for ImagineArt. Plug eight production-grade creative endpoints — image, video, music, scripts, upscaling, more — into Claude, Cursor, and any MCP client.

01

Open your client

Launch Claude Desktop and open Settings → Connectors → Add custom connector.

02

Add the server

Name it Imagine and paste the command:

npx -y @imagine/mcp-server
03

Connect & create

Click Add → Connect, sign in with your imagine.art account — and just ask Claude to generate an image.

If you're using Claude Code or Codex, the CLI works best.
